# Re:Zero — сайт-читалка

Удобная онлайн-читалка для перевода **Re:Zero**.

Здесь можно читать главы, переключаться между арками и томами, открывать дополнительные материалы, пользоваться поиском, закладками, заметками и настройками отображения текста.

## Что есть на сайте

- удобное чтение глав;
- разделение по аркам и томам;
- отдельный раздел с экстра-материалами;
- поиск по главам;
- сохранение прогресса чтения;
- закладки и заметки;
- настройки темы, размера текста, ширины страницы и межстрочного интервала;
- адаптация под ПК, планшеты и телефоны.

## Доступные материалы

На данный момент на сайте доступна **Арка 6**.

В будущем сайт может пополняться другими арками, томами и дополнительными материалами.
